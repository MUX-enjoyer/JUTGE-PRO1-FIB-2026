# Tauler d'escacs (2)

Considereu un tauler d'escacs amb $f$ files i $c$ columnes, on a cada
casella hi ha entre 0 i 9 monedes. Suposeu que la casella de dalt a
l'esquerra és blanca. Per exemple, aquest és un tauler buit amb quatre
files i cinc columnes:

::: center
:::

Feu un programa que, donat un tauler, calculi el nombre total de monedes
contingudes a les caselles blanques.

## Entrada

L'entrada comença amb el nombre de files $f$ i el nombre de columnes
$c$. Segueixen $f$ línies, cadascuna amb $c$ caràcters entre '`0`' i
'`9`'.

## Sortida

Cal escriure el nombre total de monedes a les caselles blanques del
tauler.
